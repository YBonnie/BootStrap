101教育能力平台是为网龙华渔教育业务提供所有教学活动能力的SaaS服务平台，它以“标准化 + 服务化 + 组件化”的方式全方位打造教、学、练、测、评、推荐等核心教学活动体系；通过构建丰富的教学方法库，涵盖教学设计、教学实施、教学评测、教学管理等教学全过程；同时通过快速组装教育业务应用，结合海量优质资源，开创全新教育模式与业务形态，构筑终身、开放、便捷、易用的个性化移动互联网学习平台，贯穿学前教育、基础教育、高等教育、职业教育、企业培训、非学历及终身教育等各个阶段，满足不同教育业务需求。

项目愿景

网龙华渔教育的目标是帮助更多人获得有效的教育，为世界教育的发展和进步做贡献，培养21世纪未来社会的建设者和接班人。 网龙要做的是面向未来的学习，用技术创造教育的未来。为了达成这个目标，我们统一规划，将网龙在互联网技术上的优势，全方位应用在教育上， 以平台化的方式真正实现个性化的学习、有效果的学习、情景式的学习、人性化的学习、全民终身式的学习。

核心优势

  • 打造优质教育资源分发平台

    提供安全、稳定、快速的全球化、多语种的数字化教学资源的统一管理、检索和分发能力以及服务，打造全球最大的教学软件和教育内容的分发平台，做真正面向全球，融合国家、融合学科，面向所有人的教育

  • 构建有效学习

    以学习者为中心，通过深度个性化教学方式，围绕着学习目标，智能分析适用于学习者的学习资源、学习方法。同时，借助于3D模拟、VR沉浸式、情景嵌入式等其他学习方式，充分提高学生学习兴趣，让学习更有效、更高效

  • 打造终身学习数据

    以教育数据为线索，使用户的学习数据在101教育能力平台以及所有支持ND资源标准的其他业务系统内流通，构建贯穿用户一生的学习档案，促成终生学习、促进职业发展

平台优势

  • 教育生态

    网龙华渔教育斥巨资构建聚合“软件 + 硬件 + 平台 + 资源”，开放的、可持续的教育生态圈，吸引更多合作伙伴的加入

  • 国际化、全球化

    全球最大的教学软件和内容的分发平台，以101教育PPT、101教育能力平台两大品牌为核心，做真正面向全球，融合国家、融合学科，面向所有人的教育

  • 统一帐号库（公民库）

    可绑定QQ、微信、Facebook、Google等第三方帐号，以帐号为线索，构建贯穿用户一生的学习档案，促成终生学习、促进职业发展

  • 有效学习

    以学习者为中心，围绕学习目标，智能分析适用于学习者的资源与学习方法，借助于3D模拟、VR沉浸式、情景嵌入式等学习方式，实行深度个性化教学，让学习更有效、更高效

  • 社交化学习

    人与学习内容、人与人之间连接，将被动学习转化为主动学习，形成社交互动的教育生态体系，IM+Social+养成+游戏化让学习互相督促、更加有趣

  • 教育方法论

    将部分明确、清晰的学习流程(“方法论”）放入教育引擎

  • 海量精品资源

    全学科的优质资源，与政府部门、顶尖的培训机构、名人合作，独家制作优质资源，并同步引进国内外的内容生产商的优质资源

  • VR+教育

    将沉浸式虚拟现实技术与教学相融合，为学生创设接近真实的学习环境，为学习者打造高度开发、可交互、沉浸式的三维学习环境
